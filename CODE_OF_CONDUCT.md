# Code of Conduct

## Our Pledge

We are committed to providing a friendly, safe, and welcoming environment for all contributors, regardless of experience level, gender identity, sexual orientation, disability, personal appearance, race, ethnicity, age, religion, or nationality.

## Our Standards

**Positive behavior:**
- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Gracefully accepting constructive criticism
- Focusing on what is best for the community

**Unacceptable behavior:**
- Harassment, trolling, or personal attacks
- Publishing others' private information
- Using this tool for unauthorized/illegal activities
- Any conduct that could reasonably be considered inappropriate

## Ethical Use

This project is intended for **authorized penetration testing and education only**. Contributors and users must:

- ✅ Only test systems they own or have written authorization to test
- ✅ Report vulnerabilities responsibly
- ❌ Never use this tool for unauthorized access
- ❌ Never target systems without explicit permission

## Enforcement

Violations may be reported by contacting the project maintainer. All complaints will be reviewed and investigated.
